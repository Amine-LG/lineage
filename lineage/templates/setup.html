How to run Lineage

  • Local — mock — LINEAGE_MOCK=1 python run.py. Built-in mock dataset, no cluster.
  • Local — real cluster — python run.py, using your existing oc session. See docs/quickstart.md.
  • In-cluster — build inside OpenShift, deploy as a SA. See docs/in-cluster.md.

Permissions

Read-only: get and list only, plus oc auth can-i. The only sensitive read is the HTPasswd Secret by exact name in openshift-config (optional, name-scoped). Full breakdown + least-privilege ClusterRole: docs/permissions.md.
